# 設計メモ:現実世界・文明育成シム(仮題)

**一行コンセプト**
実在の地形・地理・歴史を舞台に、まっさらな草原から物流で文明を再興し、時代を越えて街・道・鉄道が富んでいくのを「育て、眺める」サンドボックス。地球そのものが文明の発展をたどる。

**設計の軸**
SimCity 的な育成・鑑賞(建物・道路・鉄道が見た目ごと進化)× 能動的ビルダー(自分で敷設・改良する)。マイクロ管理より、意思決定と見せ場を重視。

---

## コアループ
観察(流れがある)→ 発見(どこかで詰まる)→ 解決(繋ぐ／橋／鉄道／拠点)→ 繁栄(相性の良い特産が流れ、街と網が一段進化)→ 拡張(隣接地域へ手を伸ばす)。

## 世界の初期状態
本物の地形(標高・河川・海岸・森)だけがあり、文明はゼロ。地理的に妥当な場所(水辺・低地・河川合流・史実の古い町)に小集落を点在配置。現実の現代マップは「成長の到達点＝答え合わせ」。

## 発展の単位＝地域(アーキタイプ)
山(信仰・木材)／海(漁・塩・港)／川(米・舟運)／平野(商業)… 各地域は実在の **特産品** を産出し、**名所** は特別ノード(観光・信仰・解禁条件)になる。

## ネットワークも育つ
獣道 → 土道 → 街道(石畳)→ 鉄道 → 高速／リニア。路線の格が流量の上限を決める。

## 時代 tier ＝ビジュアル進化(既存モードを活用)
草原(0)→ 江戸 → 明治 → 現実 → 未来。到達で街並み・道・空気が世代交代する。

## 飛躍条件(本作の背骨)
tier N → N+1 は、**異質なアーキタイプ**かつ **同等以上(≧)の発展段階**の地域と、**実在ルートで持続的に交易**が成立して初めて解禁。
→ 誰かが上がる前に必ず複数地域が同段階に達する「発展のピラミッド」が生まれ、自然に多地域を育てる流れになる。片方を凍結して待つ同期作業は不要。

## 歴史人物＝時代連動の偉人効果
地域の段階が人物の史実の時代に達すると「覚醒」し、史実に沿った効果を発揮。
例:太田道灌(伊勢原・築城／街道整備)、二宮尊徳(小田原・荒地再生)、伊藤博文(大磯・明治近代化)、武田信玄(甲斐・治水)、渋沢栄一(実業／金融)、伊能忠敬(測量＝マップ開拓)。

## 物流の抽象度
個体(トラック)は動かさず、辺(ルート)の流量・容量で近似(鑑賞主役のため軽量に)。

## 緊張
隘路・災害(橋の流失・土砂崩れ・低地の浸水＝海／標高と連動)でルートが切れると成長が停滞。維持も遊びのうち。

## 最小 UI
地域ごとに tier ／アーキタイプ ／「飛躍に使える相手」をハイライト。フロー表示(線の太さ＝流量・色＝健全度)。3D は現地視察・鑑賞。

## 教育的クライマックス
地理と物流の理屈だけで育てた街が、現実の地図とほぼ一致する「答え合わせ」。

## どこから始めても「地域に合った発展」を(全国対応・データ駆動)
- **原則:メカニクスは普遍、コンテンツはデータ駆動。** 仕組み(物流・tier・≧ゲート・アーキタイプ・繁栄→ビジュアル)は全国共通。地域固有(特産・名所・偉人・名前)はすべてデータから解決し、ハードコードしない。位置情報スタートと相性良く、"自分の故郷を育てる"体験になる。
- **地域の単位**:市区町村(行政界)を基本に(OSM/Wikidata に境界と属性が揃う)。
- **アーキタイプ判定(地形・地理から自動)**:海岸距離・比高(標高)・河川/低地・用途密度から、山(信仰・木材・鉱)／海(漁・港・塩)／川(米・舟運)／平野(商業)…を推定。→ 開始地点ごとに「詰まりの型」と「最初の特産」が変わる。
- **特産・名所・偉人**:Wikidata/Wikipedia の市区町村データ(名物・観光・出身者＋生没年)から供給。史実の年代で偉人が覚醒。
- **データ欠損時のフォールバック**:特産や偉人が無い地域は、アーキタイプ由来の汎用(海→漁、山→木材/信仰 等)で必ず成立させ、破綻させない。
- **開始地点の種まき**:実在の町・集落(役場所在地など)を初期集落に。地形的に妥当な場所へ配置。
- **なぜ地域ごとに違う体験になるか**:アーキタイプが開始特産と拡張方向を規定し、≧＋補完ゲートが「自分の地理に合った相手」へ手を伸ばさせる(山の町は海を、米の平野は港/市場を求める)。→ どこで始めても、その土地らしい発展路になる。
- **例(全国どこでも成立)**:北陸=米・売薬 × 北前船港／前田、瀬戸内=塩・海運 × 島嶼／村上水軍、東北=米・金 × 三陸漁港／伊達、九州=交易・砂糖 × 大陸港／島津、京=文化・工芸 × 若狭の海。
- **世界対応(将来)**:同じデータ駆動を Wikidata/OSM で海外へ拡張(コンテンツは薄くなる)。まず日本で厚く。

## 最初のスライス:相模エリア
- **大山(山／信仰)**:大山阿夫利神社＝名所、大山豆腐・大山こま＝特産、太田道灌＝偉人
- **相模川(川)**:米・舟運。渡し／橋がボトルネック
- **平塚・大磯(海)**:漁・東海道、伊藤博文(大磯)
- **小田原(海／城下)**:港・かまぼこ・梅、二宮尊徳
- **秦野(内陸)**:たばこ・名水

**最小ループ**:草原＋数集落 → 大山 × 相模川を繋ぐ → 繁栄・一段進化 → 太田道灌 覚醒 → 江戸段階 → 二宮尊徳で天井突破 → 東海道／港と交易 → 明治。
